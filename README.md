# Сайт магазина &laquo;Рево кофе&raquo;

Это HTML-проект&nbsp;&mdash; макет главной страницы магазина, продающего кофе. 
Сделан с&nbsp;использованием HTML5.

## Что внутри
&mdash;&nbsp;Шапка с&nbsp;названием 
&mdash;&nbsp;Основной блок: текст-слоган, изображение, меню навигации
&mdash;&nbsp;Раздел с&nbsp;преимуществами (магазин, качество, упаковка, помол)
&mdash;&nbsp;Каталог кофе 
&mdash;&nbsp;Блок &laquo;Подарочный набор&raquo;
&mdash;&nbsp;Комбо-наборы 

## Технологии
&mdash;&nbsp;Использование HTML5

&mdash;&nbsp;CSS: один стиль&nbsp;&mdash; `.objects { display: flex; }` &mdash; чтобы расположить карточки в&nbsp;строку
